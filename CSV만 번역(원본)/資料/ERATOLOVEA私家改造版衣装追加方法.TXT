〇単体の衣装
　※상의と하의のTEQUIP:XのXの番号が穴あきになっているため、空いてる番号から衣装を追加するようにしてください

・CLOTHES2.ERB
	@CLOTHES_LIST		：追加した衣装の分類の、繰り返し処理している箇所の数字を増やす
							例)TEQUIP:7[머리장식]にニットキャップを追加した場合は
								1196行目の「REPEAT 8」を「REPEAT 9」に修正する
									変数A = 衣装の分類 = 머리장식 = TEQUIP:7 には、8つ衣装が設定されていたが
									ニットキャップを増やしたことで、9つに増えたため「REPEAT 9」に修正する必要がある
								
							※상의、하의の衣装を追加した場合は
								1399行目の「SIF B >= 19 && B <= 24」の数字も修正しないと、
								追加した衣装が表示されないかもしれません
							
						　もし衣装の表示/非表示の制御がしたい場合も、ここに追記する

	@CLOTHES_NAME(ARG)	：追加した衣装の表示名を追記する
						　もし特定のキャラが着用した場合は別の表示名にしたい場合も、ここに追記する

・CLOTHES.ERB
	@CLOTHES_MESSAGE	：衣装を着脱した場合の地の文を変更したければ追記する

・KOJO_FUNCTION.ERBなど
	衣装関連の関数に追記の必要があれば追記する

・eraToLoveA　フラグ使用状況一覧.txt
	追加した衣装を追記する

〇セットの衣装(상의と하의がセットになった衣装)
　※상의と하의のTEQUIP:XのXは必ず同じ番号になるようにしてください
　	2021/10/04時点だと、TEQUIP:4(TEQUIP:5)の1-10と25がセット衣装のため、次にセット衣装を追加するなら26になります

・CLOTHES2.ERB
	@CLOTHES_LIST		：상의と하의の、繰り返し処理している箇所の数字を増やす
							例)ゴスロリドレスをTEQUIP:4(TEQUIP:5) = 26に追加した場合は
								1322行目と1382行目の「REPEAT 25」を「REPEAT 26」に修正する
						
						　セット衣装の하의を非表示になるように修正する
							例)ゴスロリドレスをTEQUIP:4(TEQUIP:5) = 26に追加した場合は
								1391行目「SIF (TEQUIP:상의 >= 1 && TEQUIP:상의 <= 10) || TEQUIP:상의 == 25」を
								「SIF (TEQUIP:상의 >= 1 && TEQUIP:상의 <= 10) || (TEQUIP:상의 >= 25 && TEQUIP:상의 <= 26)」に
								1395行目「SIF (B >= 1 && B <= 10) || B == 25」を
								「SIF (B >= 1 && B <= 10) || (B >= 25 && B <= 26)」に修正する
							
						　もし衣装の表示/非表示の制御がしたい場合も、ここに追記する

	@CLOTHES_NAME(ARG)	：追加した衣装の表示名を追記する(상의、하의ともに)
						　もし特定のキャラが着用した場合は別の表示名にしたい場合も、ここに追記する

・CLOTHES.ERB
	@EVENTTRAIN			：着衣設定によるセット衣装の着脱処理を追記
							「;上下セットならど下も脱げる」とコメントが書かれている箇所
								半裸(上)と半裸(下)の部分
	@CLOTHES_CHANGE		：セット衣装の着脱処理を追記
						　「;セット衣装を脱いだとき、もう片方も脱げる」とコメントが書かれている箇所(変更3箇所)と
						　「;속옷（위）を着るとき상의と전신상의が脱げる」とコメントが書かれている箇所(変更1箇所)と
						　「;セット衣装を着るとき、他のセット衣装は脱げる」とコメントが書かれている箇所(変更2箇所)
						　「;セット衣装を着たとき、もう片方も着る」とコメントが書かれている箇所(変更1箇所)
						　※「25」でファイル内検索すれば見つかると思います
	
	@PRINT_CLOTHES		：セット衣装の表示/非表示処理を追記
						　「;セット衣装を着てるとき、セット衣装(下)は表示しない」とコメントが書かれている箇所

	@CLOTHES_MESSAGE	：衣装を着脱した場合の地の文を変更したければ追記する

・KOJO_FUNCTION.ERBなど
	衣装関連の関数に追記の必要があれば追記する

・eraToLoveA　フラグ使用状況一覧.txt
	追加した衣装を追記する(상의、하의ともに)
