eraToloveA02用・토모에가타 나기나타と시즈카가타 나기나타を追加するためのデータです。

토모에가타 나기나타は陥落させやすいので、初心者向けかと思います。
「ひんむく」コマンドで、モノクルのみ残してエッチすることができます。
なので顔射すると、モノクルに精液がかかります。

시즈카가타 나기나타については、刀帳番号が12番になっています（CHARA151番以降のCSVファイルが아이に該当するため、本来であれば이마노츠루기극の刀帳番号を使いました）
토모에가타 나기나타とは逆に攻略しにくいキャラになっています。ですが、陥落させてしまえば、薙刀をはべらすことも夢ではありません。作ろうぜ！薙刀ハーレム！
