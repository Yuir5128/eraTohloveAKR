eraToLoveA用修正+追加パッチ

eraToLoveA用の、修正+追加パッチです。
ゲームをプレイするためには、別途当該ゲームの本体が必要になります。

パッチ制作者：紅


;===========================================
;パッチの適用方法
;===========================================
フォルダ内に入っているファイルを、それぞれのフォルダに移動、上書きコピーしてください。下記の修正・更新が行われます。

【仕様変更・追加】
・キャラクターNo57「아카시 쿠니유키」を追加しました。

・下記コマンドの仕様を変更しました。
「助손Ｃ애무」「助손Ｂ애무」「助손Ｖ애무」「助손Ａ애무」
「助손Ｖ挿入」「助손Ａ挿入」「助손입辱」
　これまでは、上記コマンドを選択してから、次に同コマンドを選択するまでの間、
　その行為は「継続している」とみなされ、行為が続いていましたが、
　今후は、通常の「애무」や「정상위」同様、その場限りの行為となります。
　連続で行わせたい場合は、その都度コマンドを選択する必要がありますのでご注意下さい。

※입上制作者様各位
　입上制作者様につきましては、두려움入りますが、お使いのファイルより
　「TEQUIP:助손Ｃ애무」等の記述を削除して頂けますようお願い致します。
　また、今후、입上の内容を「行為が継続している」ように見せたい場合は、
　前回コマンドを判定する「PREVCOM」等で代用してください。

・一部アイテムの購入条件を変更しました。


【不具合修正】
下記不具合を修正しました。

・「強くてニューゲーム」選択時、オープニングを表示しようとするタイミングでエラー落ちする問題
・コマンド選択時、キャラクター同士の相性判定がうまく作動しない問題
・一部のコマンド選択時、キャラクターの名前が「%CALLNAME%」等と表示されてしまい、反映されていない問題
・労役「コンサート」実行時、隊長に選択したキャラクターに、一部の経験等が反映されない問題
・コマンド「로터삽입」実行時、奴隷の처녀が失われてしまう問題
・オプションにて、「着衣設定」を「特殊（内番衣装）」に設定した際、表示がおかしくなる問題


【その他・バランス調整等】
・コマンド実行時の체력/기력消費のバランスを調整しました。
