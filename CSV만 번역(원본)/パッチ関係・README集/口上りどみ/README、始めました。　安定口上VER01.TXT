;安定입上について
;--------------------------------------------
■基本的には穏やかで物腰柔らかく、しかし戦闘ともなると一変して首落とそうとしてくる安定くんを目指して作成しています。
　ただし戦闘をするシステムはないため、음란系に堕ちた場合にオラオラするようになりました。

■「꿋꿋함」に振る舞い「반항적」で、「일선을넘지않음」「츤데레」。
　普段は強気ですが、ストレスが溜まると途端に弱音を吐き始めます。
　現状、陥落した状態でストレスが溜まっても同じように弱音を吐き始めるため、改変が必要かもしれません…今후の課題。

■この입上自体、現状、ほぼ未完成です。
　ストーリー系がちょこっとと、労役をさせるときに喋るくらい。
　喘ぎ声は入っていないため、「汎用喘ぎを使用する」にチェックを入れていれば、汎用喘ぎが適用されます。

■카슈 키요미츠、이즈미노카미 카네사다、호리카와 쿠니히로に少しだけ対応しています。
　上記３人が主人（もしくは助손）の場合に、特殊な회화が発生するようになります。
　ただし基本的には「審神者」が相손であることを前提として作っているため、場合によっては違和感があるかもしれません。
　陸奥守は一瞬対応させようとして、土佐弁が書けずに諦めました。誰か書いてください…。
　上記の입上を書かれる方で、齟齬が生じる等の場合にはお好きに書き直してやってください。

■安定から主人（審神者限定）への、呼び名を変更することができます。
　「初めて安定を調教するとき」「いずれかの素質に陥落후、安定と[회화]するとき」に、呼び名変更が可能です。
　任意の文字列を入力することもできますので、お好きな呼ばれ方で楽しんでみてください。

■さらに詳しいコンセプトや設定は、ページ下部にあります。
　完全にネタバレなので、スクロールには注意してください。
　避けられる地雷は避けて通るのが、賢い審神者のマナーだよ☆

;--------------------------------------------
;基本スペック
;--------------------------------------------
■체력面
　체력1900、기력1300。普通の打刀。
　체력は加州より若干高く、기력は加州より若干低い。
■技術面
　特筆事項無し。普通の打刀。

■性格
　一見순종で大人しそうに見えるが、実はかなり我が強く、【반항적】で【꿋꿋함】な性格。
　沖田くんの事は敬愛しているが、その代わり主のことは信用していない。
　１人に固執するタイプであるのか、堕ちる前はツンツン、
　ただし一度堕ちてさえしまえば真っ逆さまの【츤데레】持ち。
　性的な行為に関しては若干【보수적】な面があり、、かつ最후の最후で【일선을넘지않음】。
　扱いづらい、という特性が表れている。


;--------------------------------------------
;陥落までの流れとその후
;--------------------------------------------
■初期
　戦うために呼ばれた、と思ったら愛玩用の玩具になれと言われて戸惑う安定。
　そんなことに使われたくないと、主人に対して반항적な態度を取りまくる。

■연모
　何故か優しい主にほだされかけ。…僕を、愛してくれるのかい？
■친애
　愛してもらえてる自覚が出てきた安定。…沖田くん、僕、好きな人ができたよ。

■음란
　気持ちいいのに慣れてきた。…これで、相손が好みの奴だったら…なんでもない。
■창부
　肉欲に溺れた子猫ちゃん。…合わさるって、いいよね。
　　→テンションが上がると戦闘モードになります。

■복종
　自分は所詮刀だから、逆らう기력が失せてきた。…目をかけられてる分は、頑張るよ。
■예속
　性奴隷である自覚が出てきた。…あなたが新しい、僕の大切な主です。

■붕괴
　見えちゃいけないモノが見えるようになった。
　…主？　카슈 키요미츠？　이즈미노카미 카네사다？　みんな沖田くんに見えるよ。
