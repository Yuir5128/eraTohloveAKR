;===========================================
;패치의 적용 방법
;===========================================
폴더내에 들어가 있는 파일을,각각의 폴더에 이동,덧쓰기 카피해 주세요. 아래와 같은 수정·갱신을 합니다.

【사양 변경·추가】
·캐릭터 No57 「아카시 쿠니유키」를 추가했습니다.

·아래와 같은 커맨드의 사양을 변경했습니다.
「조수C애무」 「조수B애무」 「조수V애무」 「조수A애무」
「조수V 삽입」 「조수A 삽입」 「조수입욕」
　이것까지는,상기 커맨드를 선택하고 나서,다음에 동커맨드를 선택할 때까지의 사이,
　그 행위는 「계속하고 있다」라고 보여 행위가 계속되고 있었습니다만,
　지금후는,통상의 「애무」나 「정상위」같이,그때 뿐의 행위가 됩니다.
　연속으로 실시하게 하고 싶은 경우는,그때마다 커맨드를 선택할 필요가 있기 때문에 주의해 주세요.

※대사 제작자님 여러분
　대사 제작자님에 대해서는,두려움 들어갑니다만,사용의 파일보다
　「TEQUIP:조수C애무」등의 기술을 삭제해 받을 수 있도록 부탁드립니다.
　또,지금후,대사의 내용을 「행위가 계속하고 있다」같게 보여 주고 싶은 경우는,
　전회 커맨드를 판정하는 「PREVCOM」등으로 대용해 주세요.

·일부 아이템의 구입 조건을 변경했습니다.


【불편 수정】
아래와 같이 불편을 수정했습니다.

·「강해서 뉴 게임」선택시,오프닝을 표시하려고 하는 타이밍에 에러 초월하는 문제
·커맨드 선택시,캐릭터끼리의 궁합 판정이 잘 작동하지 않는 문제
·일부의 커맨드 선택시,캐릭터의 이름이 「%CALLNAME%」등으로 표시되어 버려,반영되어 있지 않은 문제
·노역 「콘서트」실행시,대장에게 선택한 캐릭터에,일부의 경험등이 반영되지 않는 문제
·커맨드 「로터삽입」실행시,노예의처녀가 없어져 버리는 문제
·옵션에서,「착의 설정」을 「특수(내번의상)」로 설정했을 때,표시가 이상해지는 문제
