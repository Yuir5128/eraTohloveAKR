●경장 관련
----------------------------------------------------
·초기 착의 설정에 「특수(경장)」를 추가했습니다
	초기 착의 설정은 콘피그로부터 변경할 수 있습니다

·경장을 건네주지 않은 상태로,초기 착의 설정을 「특수(경장)」로 했을 경우
	통상 의상을 전착(상하 어느쪽이나 입고 있는 상태)이 됩니다

·커맨드 「선물」로부터 경장을 건네줄 수 있게 되었습니다
	함락이 끝난 상태 또한 소지 코슈금이 5만고슈금이상 없으면 건네줄 수 없습니다
	CFLAG:614를 사용합니다

【중요】촛대절대사을 도입하고 있는 경우
	KOJO_커맨드_K73.ERB의 「201,선물 CFLAG:403」에
	이하의 3점을 덧붙여 씀 하지 않으면 경장을 건네줄 수 없습니다
		-----2800행째근처 여기로부터-----
		;경장은 함락이 끝난 상태가 아니면 건네줄 수 없다
		SIF MONEY >= 50000 && KANRAKU(TARGET)
			PRINTFORML [10] 경장(50000고슈금)
		-----2802행째근처 여기까지-----
		
		-----2859행째근처 여기로부터-----
		ELSEIF RESULT == 10 && MONEY > 50000 && KANRAKU(TARGET)
			;경장을선물
			TFLAG:17 = 11
			MONEY -= 50000
			;경장입손플래그를 온
			CFLAG:614 = 1
		-----2864행째근처 여기까지-----
		
		-----3980행째근처 여기로부터-----
		ELSEIF TFLAG:17 == 11
			PRINTFORM 경장
		-----3981행째근처 여기로부터-----

· 각 캐릭터의 경장시의 의상 설정을 추가했습니다
	공식에서 공개 끝난 겐지 형제까지 설정되어 있을 것입니다
		아직 공개되어 있지 않은 캐릭터도 유카타를 몸에 대게 되어 있습니다

선언했는데 미안합니다

●처녀·동정·백버진를 빼앗은 상대의 표시가
실제로 빼앗은 상대와 다른 경우가 있다
----------------------------------------------------
※사양인 것이나 버그인 것이나 판별이 되지 않기 때문에 고치고 있지 않습니다

처녀·동정·백버진를 빼앗은 상대는
빼앗긴 캐릭터의 CSTR:10(처녀) CSTR:11(동정) CSTR:12(백버진)에 기록되고 있습니다
(TRACHECK.ERB 에 해당 처리 부분)

그러나,이하와 같은 상황이라면 실제로 빼앗은 상대와 다른 상대가 기록되는 일이 있는 것 같습니다
예) 조교에 대해,조교자가 조수에 교대하고 있는 상황으로
조교자(조수)가 삽입 커맨드에 의해 노예의처녀·동정·백버진를 빼앗았을 경우
기록되는 이름이 마스터의 이름이 된다

마스터가 조수에 명령해 노예의처녀·동정·백버진를 빼앗게 했다,라고 생각할 수 없는 것도 아니기 때문에
사양인 것인가,버그인 것인가의 판별이 되지 않고 고치고 있지 않습니다

※거치고 싶고 실마리응의 안면 플래그는,상기의 예에서는 노예의처녀·동정·백버진를 빼앗은 상대로서
조수를 기록하고 있다

※마스터가 조교자인 채,조수삽입명령커맨드에 의해 노예의처녀·동정·백버진를 빼앗았을 경우
기록되는 이름은 조수의 이름
